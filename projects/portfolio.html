Personal Portfolio

Web Development Fundamentals

View on GitHub Live Demo

Overview

A fully responsive portfolio website demonstrating web development fundamentals. Built with vanilla HTML, CSS, and JavaScript—no frameworks. Features a clean design system, dark/light theme, and responsive mobile-first layout.

Tech Stack

HTML5 CSS3 Vanilla JavaScript Responsive Design CSS Variables Web APIs

Key Features

  • Design system with CSS custom properties
  • Dark/light theme with localStorage persistence
  • Fully responsive mobile-first layout
  • Smooth scroll with fixed navigation offset
  • SEO-optimized and accessible
  • Performance-optimized animations

Problem & Solution

Problem: Need a professional online presence that showcases web development skills and demonstrates vanilla JavaScript expertise without relying on frameworks.

Solution: Built a fast, accessible portfolio using modular CSS architecture, semantic HTML, and performance-optimized JavaScript. The design system ensures consistency and maintainability.

Learning Outcomes

  • CSS architecture and design system fundamentals
  • Responsive design patterns and mobile-first approach
  • Web APIs: localStorage, IntersectionObserver, theme management
  • Accessibility best practices and semantic HTML
  • Performance optimization techniques
